================================================================================
HOW TO START THE PROJECT - QUICK START GUIDE
================================================================================

STEP 1: Open Terminal/PowerShell
Navigate to your project directory:
cd C:\Users\usuario\Desktop\Mircro1AI

STEP 2: Start the Server
Run this command:

python -m uvicorn generativeai:app --reload --host 0.0.0.0 --port 8000

EXPLANATION:
- python -m uvicorn    : Run uvicorn module
- generativeai:app     : Load 'app' from 'generativeai.py' file
- --reload             : Auto-restart when code changes
- --host 0.0.0.0       : Accept connections from any IP
- --port 8000          : Run on port 8000

STEP 3: Access the API
Once you see "Application startup complete", open your browser:

Interactive Docs: http://localhost:8000/docs
Alternative Docs: http://localhost:8000/redoc
Health Check:     http://localhost:8000/

STEP 4: Stop the Server
Press CTRL+C in the terminal

================================================================================
ALTERNATIVE PORTS
================================================================================

Port 8001:
python -m uvicorn generativeai:app --reload --host 0.0.0.0 --port 8001
Then visit: http://localhost:8001/docs

Port 8080:
python -m uvicorn generativeai:app --reload --host 0.0.0.0 --port 8080
Then visit: http://localhost:8080/docs

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "python: command not found"
Solution: Use full path or check Python installation
Try: python3 -m uvicorn generativeai:app --reload --host 0.0.0.0 --port 8000

Problem: "No module named 'uvicorn'"
Solution: Install dependencies first
pip install fastapi uvicorn pydantic

Problem: "Address already in use"
Solution: Port is busy, use different port or kill existing process
Use port 8001 instead: --port 8001

Problem: "Error loading ASGI app"
Solution: Make sure you're in the correct directory with generativeai.py
Check: dir (should show generativeai.py)

================================================================================
PRODUCTION VS DEVELOPMENT
================================================================================

DEVELOPMENT (what we use):
python -m uvicorn generativeai:app --reload --host 0.0.0.0 --port 8000
- Auto-reloads on code changes
- Detailed error messages
- Single worker

PRODUCTION (for deployment):
python -m uvicorn generativeai:app --host 0.0.0.0 --port 8000 --workers 4
- No auto-reload
- Multiple workers
- Better performance
- Remove --reload flag

================================================================================
BACKGROUND MODE (Optional)
================================================================================

If you want to run in background and use the same terminal:

PowerShell:
Start-Job -ScriptBlock { 
  Set-Location "C:\Users\usuario\Desktop\Mircro1AI"
  python -m uvicorn generativeai:app --host 0.0.0.0 --port 8000 
}

To stop background job:
Get-Job | Stop-Job
Get-Job | Remove-Job

Note: Logs won't be visible in background mode.
Recommended: Run in foreground for development.
